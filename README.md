# fcs-website2

A responsive, image-driven portfolio website for Forest Chorus Studio, an embroidery design artist. Built with React, Tailwind CSS, and Framer Motion.

## Features

- Dynamic image grid gallery
- Lightbox for detailed image viewing
- Hero section with full-screen image carousel
- Contact form
- Sections for magazine features, published book, Etsy links, testimonials, and licensing info
- SEO optimization (meta tags, alt text, sitemap.xml)
- Modular and well-documented code

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ty-stackhouse/fcs-website2.git
   ```
2. Install dependencies:
   ```bash
   cd fcs-website2
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

The site can be deployed to platforms like Vercel or Netlify. A GitHub Actions workflow will be set up for CI/CD.

## Technologies

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite
